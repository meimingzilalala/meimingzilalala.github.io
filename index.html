字母表有非空性、有穷性、单一性

ε代表空串
{ε}代表仅含有空串的集合
Φ代表空集
αβ代表两个字符串α与β的连接（并置）
αn 代表α的n次连接，其中α0 =ε，αn =αn-1 α 其中n>0
AB代表集合A与B的连接，A={a1,a2,a3,…,an}，B={b1,b2,b3,…,bm}

下载nodejs和cnpm

直接在nodejs官网进行下载并安装
node -v命令查看node的版本
npm -v命令查看npm的版本
然后利用npm安装淘宝的cnpm，下载速度快一些
npm install -g cnpm --registry=http://registry.npm.taobao.org命令安装淘宝的cnpm包管理器
cnpm -v命令查看cnpm的版本
